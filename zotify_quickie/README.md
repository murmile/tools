zotify_quickie.sh is a lightweight CLI tool for quick access to an already set up and running Zotify installation. It allows you to quickly paste Spotify links into CLI to download them during DJ sets or for sampling during music production

1. Pre-install:
	- python
	- ffmpeg
	- zotify
2. Run "zotify" in terminal to configure username, password and download path
3. Run the tool in CLI and enjoy
